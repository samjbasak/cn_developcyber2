  1. RANSOMWARE: Is a rogue software that prevents the normal operation of a website. It can slow down or even shutdown the website and it will not work until a ransom is paid.
  2. MALEWARE: Menacing software that is intended to damage or disable websites.
  3. ACCIDENTAL FILE DELETION: Website can be taken down by innocent mistake of a user. The employee may rename a file used for the website or even delete it. This will render some part or put out of action, an entire website. These are easy to spot and fix.
  4. DELIBERATE FILE DELETION: Hackers can deliberately delete, add or modify files used by the website which will compromise the live website. One can reply on backups to restore the original files.
  5. EMPLOYEE SABATOGUE: Disgruntled employee can e-sabatoge by outdated username and password that administrative oversights did not disable upon the employee leaving. An unhappy worker or former employee can take down a website and/or steal critical company data. Entire websites would be at risk for data theft or sabotage.
  6. DDoS ATTACK: In October 2016, several large online companies such as PayPal, Netflix and PlayStation Network suffered DDoS attacks. The hosting provider Dyn was flooded with network traffic which caused live website operation failure. The bots flood bandwidth of targeted websites servers preventing legitimate traffic from viewing websites.
